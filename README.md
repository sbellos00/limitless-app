# Limitless

Personal daily operating system. Tracks morning ritual, creative blocks, deep work sessions, night routine, mental fitness, badges, votes, dopamine, nutrition, sleep.

## Quick Start

```bash
npm install
npm run dev:all    # Vite (3002) + Express (3001)
```
